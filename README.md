# GerenciamentoBiblioteca

Sistema de gerenciamento de biblioteca desenvolvido para controlar livros, usuários, empréstimos, devoluções e multas.

## Objetivo

O objetivo do sistema é facilitar a administração de uma biblioteca, permitindo que administradores e bibliotecários acompanhem o acervo, registrem empréstimos, controlem prazos de devolução e gerenciem multas por atraso.

## Funcionalidades

- Cadastro e listagem de livros
- Cadastro e gerenciamento de usuários
- Registro de empréstimos
- Registro de devoluções
- Cálculo de multas por atraso
- Dashboard com indicadores atualizados
- Alertas de devolução próxima ao prazo
- Envio de aviso via WhatsApp
- Controle de acesso para administrador e bibliotecário

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Gradle
- HTML
- CSS
- JavaScript
